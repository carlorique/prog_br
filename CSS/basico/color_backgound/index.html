Color

Observe que a lista de valores de cores aceitas foi estendida à medida que a especificação
evoluiu, culminando com as cores CSS3 mais recentes.

Associado à cor no espaço sRGB, um valor "color" também pode consistir de uma coordenada
de valor de transparência canal-alfa, Indicando como a cor deve se composta com sua cor de fundo.

Embora os valores de cores CSS sejam definidos com precisão, eles podem aparecer de forma
diferente em dispositivos de saída diferentes. A maioria deles não está calibrada e alguns
navegadores não suportam o perfil de cores dos dispositivos de saída. Sem estes, a
renderização de cores pode variar muito.

Background

A background-colorpropriedade define a cor de fundo de um elemento.
O plano de fundo de um elemento é o tamanho total do elemento,
incluindo preenchimento e borda (mas não a margem).
Dica: Use uma cor de fundo e uma cor de texto
que facilite a leitura do texto.
